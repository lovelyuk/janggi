# 웹 장기 (1인용)

간단한 웹 페이지에서 즐길 수 있는 1인용 장기 게임입니다. 플레이어는 홍 진영을
조종하며, 청 진영은 기본적인 탐색 알고리즘을 사용하는 AI가 두게 됩니다.

## 실행 방법

1. 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저(Chrome, Edge, Firefox 등)로 열면 바로 게임을
   시작할 수 있습니다.

별도의 빌드나 서버 실행이 필요하지 않습니다.

## 주요 기능

- 전통 장기 말 배치 및 이동 규칙 구현 (장군, 사, 상, 마, 차, 포, 병/졸)
- 장군 멍군 규칙과 포의 뛰어넘기, 말과 상의 길막 등 기본 룰 반영
- 간단한 미니맥스 기반 AI로 1인용 대전 지원
- 선택한 말의 이동 가능 위치 하이라이팅 및 최근 이동 표시
- 장군 상태 표시 및 새 게임 초기화 버튼 제공

## 개발 노트

- 게임 로직은 `script.js`에 구현되어 있으며, CSS는 `styles.css`에 포함돼 있습니다.
- 브라우저 호환성을 위해 외부 의존성을 사용하지 않았습니다.
- 더 깊은 난이도의 AI나 추가 편의 기능은 자유롭게 확장해 보세요.
